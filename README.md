Ansible playbook/roles for setup of workstation


# Install Ansible and other dependencies

All OSes requires an ssh key setup and working with github


## macos
```
sh setupmac.sh
```

## Arch Linux
```
pacman -S ansible

ansible-playbook install.yml
```
